# akdl
Amazing Keyword Definition Language: helps creating your syntax

It allows you to generate your parsing and runtime code out of a semi-formal definition of its syntax.

To run it:
	java -cp <path to the jar>akdl.jar  akdl.AKDL  <your definition file>

Your definition file should have “.def” extension.

If you are not familiar with the notation AKDL uses for defining your syntax, please refer to the associated tool called “Visual AKDL Editor” you can also find
in GitHub.
